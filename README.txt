Running the app on multiple computers
    - Make sure all machines are in same network
    - Get your IPv4 Address using "ipconfig" command on the terminal
        - Example IPv4 192.168.43.152
        - Consider "IPv4" not "Default Gateway"
    - Type "runall" in terminal and hit enter
    - "Allow access" if assked on a propmt
    - Now open browser on another machine and type the IPv4 address followed by a :8000 port number
        Example: 192.168.43.152:8000
